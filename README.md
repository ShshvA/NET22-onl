#### **Основное задание**

Написать программу для парсинга JSON документа.

Программа на вход получает строку к папке, где находится документ.  
Распарсить нужно только один документа - соответственно, предусмотреть различные проверки, например на то, что в папке нет файлов, или в папке несколько документов формата JSON и другие возможные проверки.  
Необходимо распарсить JSON документ в объект Squad, а затем сохранить в виде xml.
Название файла для записи должно состоять из значений тегов и имеет вид: <squadName>.xml.
