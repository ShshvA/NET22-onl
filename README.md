# TMS Lesson 4 - Homework

---

### Циклы

Число N прочитать из Console.ReadLine

1. Необходимо вывести на экран числа от N до 1.
2. Необходимо, чтоб программа выводила на экран вот такую последовательность : 7 14 21 28 42 49….. 
3. Вывод последовательности Фибоначчи

---

### Матрицы
Создать программу работы с матрицами (двухмерными массивами) c возможностью выбора размера(n, m < 6).

Заполните их автоматически. 
Для генерации случайного числа используйте метод Math.random()(-9 до 9).

Добавьте меню.

1. Вывести матрицу на консоль (добавить оформление, чтобы выглядело как матрица)
2. Найти количество положительных/отрицательных чисел в матрице.(можно использовать две переменные)
3. Вывести матрицу так чтобы в четных строках, отображались только четные элементы, а в нечетных – нечетные. Если элементов нет, пустая строка.
4. Используя словарь подсчитать и вывести сколько раз, какое число было в матрице.(Словарь должен содержать только такие числа, которые есть в матрице)

### Листы
1. Создать лист целых чисел. Заполнить вручную или Math.random(). 
2. Дать возможность вывода, добавления или удаления элементов.
3. Заменить каждый четный элемент на удвоенное значение (*2), а нечетный на 0.Вывести лист.
4. По этому листу создать HashSet. Вывести.
